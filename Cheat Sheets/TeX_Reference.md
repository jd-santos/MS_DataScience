# TeX Reference
- Begin and end statements with $ $
- Fraction
	- \frac{Numerator}{Denominator}
		$\frac{Numerator}{Denominator}$
- Mean/Overline
	- \overline{X}
		$\overline{X}$
- Superscript
	- x^{superscript}
	- $x^{superscript}$
- Subscript
	- x_{subscript}
	- $x_{subscript}$
- Greater/Less than or equal to
	- \geq
	- \leq
	- $\geq$
	- $\leq$
